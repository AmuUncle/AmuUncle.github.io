2、安装selenium
2.1、通过pip 安装
pip install selenium

2.2、通过下载包安装 ,直接下载selenium包：
https://pypi.python.org/pypi/selenium
解压，cmd进入目录:
python setup.py install

3 、 安装Chrome driver
下载地址：http://npm.taobao.org/mirrors/chromedriver
