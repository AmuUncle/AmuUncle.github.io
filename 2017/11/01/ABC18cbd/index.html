1. 新建项目

属性 修改值 解释
ID IDC_EDIT_TXT 控件ID
Want Return TRUE 允许回车
Multiline TRUE 允许多行
Horizontal Scoll TRUE 水平滚动条
Vertical Scoll TRUE 垂直滚动条
